# Statistical Functions

| Function | Description |
|---|---|
| **varS**(array) | Returns the sample variance of the numeric array. <br/>Uses `n − 1` as the denominator (Bessel's correction). <br/>Returns `null` if fewer than two non-null values are present. <br/>**Example:** <br/>`varS(3, 4, 8)` <br/>returns <br/>`7.0` |
| **varP**(array) | Returns the population variance of the numeric array. <br/>Uses `n` as the denominator. <br/>Returns `0` when the array contains exactly one non-null value. <br/>**Example:** <br/>`varP(3, 4, 8)` <br/>returns <br/>`4.666666666666667` |
| **stdevS**(array) | Returns the sample standard deviation of the numeric array. <br/>Equivalent to the square root of `varS`. <br/>Returns `null` if fewer than two non-null values are present. <br/>**Example:** <br/>`stdevS(3, 4, 8)` <br/>returns <br/>`2.6457513110645907` |
| **stdevP**(array) | Returns the population standard deviation of the numeric array. <br/>Equivalent to the square root of `varP`. <br/>**Example:** <br/>`stdevP(3, 4, 8)` <br/>returns <br/>`2.160246899469287` |
| **covarS**(y, x) | Returns the sample covariance between arrays `y` and `x`. <br/>Uses `n − 1` as the denominator. <br/>Pairs where either element is `null` are excluded. <br/>Returns `null` if the arrays differ in length, are empty, or contain fewer than two non-null pairs. <br/>**Example:** <br/>`covarS({3, 4, 8}, {3, 10, 8})` <br/>returns <br/>`4.0` |
| **covarP**(y, x) | Returns the population covariance between arrays `y` and `x`. <br/>Uses `n` as the denominator. <br/>Pairs where either element is `null` are excluded. <br/>Returns `null` if the arrays differ in length or are empty. <br/>**Example:** <br/>`covarP({3, 4, 8}, {3, 10, 8})` <br/>returns <br/>`2.6666666666666665` |
| **correl**(y, x) | Returns the Pearson correlation coefficient between arrays `y` and `x`. <br/>Values range from `−1` (perfect negative correlation) to `1` (perfect positive correlation). <br/>Returns `null` if the arrays differ in length, are empty, or if either variable has zero variance. <br/>**Example:** <br/>`correl({3, 4, 8}, {3, 10, 8})` <br/>returns <br/>`0.4193139346887673` |
| **rsq**(y, x) | Returns the coefficient of determination (R²) between arrays `y` and `x`. <br/>Calculated as the square of `correl`. <br/>Values range from `0` to `1` and represent the proportion of variance in `y` explained by `x`. <br/>**Example:** <br/>`rsq({3, 4, 8}, {3, 10, 8})` <br/>returns <br/>`0.17582417582417584` |
| **slope**(knownY, knownX) | Returns the slope of the least-squares regression line fitted to `knownY` and `knownX`. <br/>Represents the change in `Y` for a one-unit increase in `X`. <br/>Returns `null` if the arrays differ in length, contain fewer than two non-null pairs, or if `knownX` has zero variance. <br/>**Example:** <br/>`slope({3, 4, 8}, {3, 10, 8})` <br/>returns <br/>`0.3076923076923077` |
| **intercept**(knownY, knownX) | Returns the Y-intercept of the least-squares regression line fitted to `knownY` and `knownX`. <br/>Represents the predicted value of `Y` when `X` equals zero. <br/>Requires at least two non-null data point pairs. <br/>**Example:** <br/>`intercept({3, 4, 8}, {3, 10, 8})` <br/>returns <br/>`2.846153846153846` |
| **forecast**(x, knownY, knownX) | Returns the predicted Y value for a given `x` using the linear regression line fitted to `knownY` and `knownX`. <br/>Equivalent to `intercept(knownY, knownX) + slope(knownY, knownX) * x`. <br/>Returns `null` if `x` is `null` or if the regression line cannot be determined. <br/>**Example:** <br/>`forecast(1, {3, 4, 8}, {3, 10, 8})` <br/>returns <br/>`3.1538461538461533` |
