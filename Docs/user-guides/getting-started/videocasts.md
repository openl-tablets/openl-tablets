## Overview of OpenL Tablets self-service infrastructure

This videocast introduces the OpenL Studio application (version 5.11) and shows how an analyst can modify a rules
project. The example used is an insurance rating plan. The videocast also introduces the Rules Repository, which stores
rules projects, and demonstrates how to edit project rules using either the OpenL Studio Rules Editor or Microsoft
Excel.

## Basics of rules tables: Decision tables

This videocast introduces the basic concepts of OpenL Tablets business rules and their processing. It provides a brief
overview of all rule table types supported by OpenL, explains the syntax required to develop rules in the form of
decision tables, and includes examples and a demonstration of rule development for this table type.

## Datatype tables — defining the domain model

This videocast explains what datatyping is and its role in creating business rules. It covers the domain (data) model of
a rules project and how to develop it using datatype table types. You will also learn about predefined data types
supported by OpenL and how to create your own project vocabulary. The videocast concludes with a demonstration of
creating datatype and data tables using the OpenL Studio tool.

<p><iframe src="https://www.youtube.com/embed/Iew_cXOXikM" frameborder="0" width="420" height="315" allowfullscreen></iframe></p>

[Watch on YouTube](https://youtu.be/Iew_cXOXikM)

## Development of Calculation Rules: Spreadsheet Tables

Business rules vary in nature: beyond simple decisions, many require calculation logic and formulas. This videocast
reviews how to develop calculation rules using OpenL Spreadsheet Tables. You will learn the basics of spreadsheet rules
and the required syntax, see examples, and watch a demonstration of rule tracing functionality in OpenL Studio. Test
tables are discussed as well.
