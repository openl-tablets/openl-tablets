OpenL Tablets **5.16.4** is a patch release with improvements and a bug fix.

## Improvements

* Added the ability to register a service update listener.
* Enhanced WADL annotation display for default `Date` values.

## Bug Fixes

* Fixed: Incorrect default date value generation in Java classes.
