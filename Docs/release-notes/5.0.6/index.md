OpenL Tablets **5.0.6** is a patch release with bug fixes.

## Bug Fixes

* Fixed: Saving non-string cells in the Table Editor caused formatting issues.
* Fixed: Repository and Web Service deployment demo configuration issues.
* Fixed: Optimized Decision Tables had problems when variable names were misspelled or used incorrect casing.
