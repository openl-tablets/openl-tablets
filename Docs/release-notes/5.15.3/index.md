OpenL Tablets **5.15.3** is a patch release with improvements and bug fixes.

## Improvements

* Java 8 compatibility support added.
* Upgraded POI to 3.12.
* Streamlined property customization in the Demo package initialization.

## Bug Fixes

* Fixed: Color display issues within the table editor.
* Fixed: The `OpenL Simple Project` archetype functionality.
