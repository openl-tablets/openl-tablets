OpenL Tablets **5.17.1** is a patch release with improvements and bug fixes.

## Improvements

* Removed Log4j dependency from Core.
* Improved default Ehcache configuration.

## Bug Fixes

* Fixed: Logging to Cassandra of POST requests in REST services.
* Fixed: Reload of service after multi-modules project redeploy.
* Fixed: JavaScript error in the Admin page.
* Fixed: Links to documentation.
