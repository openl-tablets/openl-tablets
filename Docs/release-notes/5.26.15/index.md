OpenL Tablets **5.26.15** is a maintenance release with Rule Services improvements and library updates for improved security and stability.

## Improvements

### Rule Services

* `JsonUtils` configured to use `yyyy-MM-dd'T'HH:mm:ss.SSS` date format by default.
* New capability to invoke `OpenLService` using JSON string arguments similar to REST services.
