OpenL Tablets **5.19.2** is a patch release with an improvement and bug fixes.

## Improvements

* Implemented validation for replacing the main artifact in `openl-maven-plugin`.

## Bug Fixes

* Fixed: Inaccurate type list in the Datatype creation wizard.
* Fixed: Casting error affecting arrays containing null values.
