# Images for OpenL Tablets 6.0.0 Release Notes

Upload screenshots and diagrams to this folder to reference in the release notes.

## Naming Convention
- Use descriptive names: `feature-name.png` not `screenshot1.png`
- Use lowercase with hyphens
- Prefer PNG format for screenshots
- Keep file sizes reasonable (compress if needed)

## Referenced Images

The following images are referenced in the release notes and need to be uploaded:

- `admin-panel.png` — Screenshot of the redesigned Admin Panel showing the unified navigation and user management interface
- `mcp-integration.png` — Diagram or screenshot illustrating the MCP integration architecture and AI assistant workflow
