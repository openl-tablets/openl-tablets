OpenL Tablets **5.17.3** is a patch release with improvements and a bug fix.

## Improvements

* Implemented additional web service redeploy process for failure scenarios.
* Integrated Active Directory support within WebStudio.

## Bug Fixes

* Fixed: HTTP 500 error occurring during WebStudio login attempts following session timeout.
