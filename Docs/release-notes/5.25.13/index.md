OpenL Tablets **5.25.13** is a maintenance release that improves exception messaging in Rule Services and fixes several issues in the core engine.

## Contents

* [Improvements](#improvements)
* [Bug Fixes](#bug-fixes)

## Improvements

### Rule Services

* Exception messages now provide more informative details when Rule Services encounters failures.

## Bug Fixes

### Core

* Fixed incorrect text highlighting occurring when rule names begin with numerals.
* Fixed unhelpful error messages appearing when OpenL Tablets keywords are misused within tables.
