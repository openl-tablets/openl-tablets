OpenL Tablets **5.15.1** is a patch release with performance improvements.

## Improvements

**Core:**

* Performance improvements for versioning by business dimensions.

**Web Services:**

* Performance improvements in lazy loading functionality.

**Web Services & WebStudio:**

* Added property dispatching validation that can be switched on/off for rules overloading validation.
