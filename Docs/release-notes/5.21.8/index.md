OpenL Tablets **5.21.8** is a patch release containing an improvement and a bug fix.

## Improvements

**WebStudio:**

* Resolved an unintended project locking issue that occurred when users accidentally enabled "In editing" mode.

## Bug Fixes

**Core:**

* Fixed: Performance degradation in `ObjectToDataOpenCastConvertor`.
