OpenL Tablets **5.26.10** is a maintenance release introducing access to table properties, reduced memory usage in spreadsheet tables, security fixes, and library updates.

## New Features

### Core

* Access to the properties of the tables and all tables with the same signature.

## Improvements

### Core

* Memory usage has been reduced by spreadsheet tables.

### WebStudio

* All null values are now included in the Excel file generated by the "Run into File" function.

## Bug Fixes

### Rule Services

* Fixed path traversal in code sink vulnerability.
* Fixed charset missing in the response if the return type is `text/plain`.
* Fixed OpenAPI generation behind the reverse proxy.

### WebStudio

* Fixed incorrect table name displayed in the table tree if the table title contains a line break.
