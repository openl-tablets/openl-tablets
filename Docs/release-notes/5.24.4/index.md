OpenL Tablets **5.24.4** is a maintenance release that introduces new JSON object operations for improved performance
and fixes a Maven build failure.

## Improvements

### Core

* Implemented new operations to make working with JSON objects faster.

## Bug Fixes

### Maven Plugin

* Fixed build failure in Maven for projects with four dependencies.
