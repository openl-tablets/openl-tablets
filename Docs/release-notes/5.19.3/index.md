title: OpenL Tablets 5.19.3 Release Notes
date: 2017-05-17
description: Adds support for Java keywords in rule parameters and fixes interface generation with arrays in the Maven plugin.
---

OpenL Tablets **5.19.3** is a patch release with an improvement and a bug fix.

## Improvements

* Java keywords are now supported in rule parameters.

## Bug Fixes

* Fixed: `openl-maven-plugin` did not allow generating an interface with arrays.
