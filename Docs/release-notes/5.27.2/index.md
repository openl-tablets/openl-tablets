OpenL Tablets **5.27.2** is a maintenance release delivering targeted Rule Services improvements.

## Improvements

* Optimized logging to capture only OpenL method arguments and result objects, reducing the verbosity of logs.
* Integrated `Accept-Language` header support in the OpenAPI schema.
* Disabled logging of OpenL settings on startup, enhancing security posture.
