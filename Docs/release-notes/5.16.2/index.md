OpenL Tablets **5.16.2** is a patch release with bug fixes and a dependency cleanup.

## Improvements

* Removed the `org.apache.ant` dependency from the rules project.

## Bug Fixes

* Fixed: Production repository initialization in the database for the Demo Package.
* Fixed: Missing data from the production repository stored in MS SQL database.
