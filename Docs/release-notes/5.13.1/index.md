OpenL Tablets **5.13.1** is a patch release with Demo Package improvements and WebStudio bug fixes.

## Improvements

**Demo Package:**

* Improved the home page by adding steps showing how to see rule changes in action.
* Enabled GZIP compression of HTTP traffic.

## Bug Fixes

**WebStudio:**

* Fixed: Projects created via XLS/XLSX file uploads in the Repository had incorrect revision and metadata properties in
  IE9/IE10.
* Fixed: Links to guides do not work.
* Fixed: Display of the file name description icon under Firefox, IE9, and IE10.
