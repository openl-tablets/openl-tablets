OpenL Tablets **5.15.2** is a patch release with improvements and bug fixes.

## Improvements

**Core:**

* New Canada properties for versioning added.

**Web Services:**

* Converter to calculation steps DTOs now supports additional scenarios without requiring configuration.

## Bug Fixes

**Core:**

* Fixed: Array of `Class` objects in Data tables.

**Web Services:**

* Fixed: Dynamic interface now works properly with the converter to calculation steps.
