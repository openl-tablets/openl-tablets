OpenL Tablets **5.23.7** is a patch release containing bug fixes.

## Bug Fixes

**Core:**

* Fixed: Lookup with more than 100 columns returns incorrect values in some cases.
* Fixed: The error "Invalid Decision Table header" appears in SmartRule.
