OpenL Tablets **5.10.1** is a patch release with improvements and bug fixes.

## Improvements

**WebStudio:**

* WebSphere Application Server 8.5 support.
* Backward compatibility with Web Services of version 5.9.3 and earlier.

## Bug Fixes

**Core:**

* Fixed: Multiple memory leak issues.
* Fixed: Mapping issue specific to OpenL Tablets 5.10.0.

**WebStudio:**

* Fixed: Formula conversion from OpenL Tablets to Excel format.
* Fixed: WebDAV connection setup configuration issue.
* Fixed: Validation logic in the Alias Datatype table wizard.
* Fixed: Default path assignment in the Administration section.
