OpenL Tablets **4.1.1** is a feature release with WebStudio customization, benchmarking tools, and performance
improvements.

## Contents

* [New Features](#new-features)
* [Improvements](#improvements)
* [Bug Fixes](#bug-fixes)

## New Features

### WebStudio Start Screen Customization

Simple customizations can be applied to the WebStudio start screen using three system properties to configure the header
image, an intro HTML or JSP file, and the project tree background color.

### Interactive Benchmarks

Interactive benchmarks are now available for all methods, enabling users to run and compare performance measurements.
Tutorial 5 is added to demonstrate benchmarking.

## Improvements

* Decision Table performance is enhanced through indexing for specific condition types, delivering notable gains for
  larger tables.

## Bug Fixes

* Fixed: The "Simple OpenL Tablets Project" template contained an outdated reference; resolved by running the "Generate
  Wrapper..." script in Eclipse.
