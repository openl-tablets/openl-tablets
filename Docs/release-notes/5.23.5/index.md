OpenL Tablets **5.23.5** is a maintenance release containing an improvement and bug fixes.

## Improvements

* Generating `MANIFEST.MF` on deploy from WebStudio.

## Bug Fixes

* Fixed: Parent datatype is displayed instead of the extended datatype in WebStudio on the Run and Trace UI.
* Fixed: Failure of applications that were started after Rule Services on JBoss 7.2.
* Fixed: Information about subtypes of the datatype is not presented in JSON.
