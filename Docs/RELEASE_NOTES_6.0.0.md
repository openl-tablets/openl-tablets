# OpenL Tablets Release Notes

## Version: 6.0.0-SNAPSHOT
**Release Date:** January 22, 2026

## 🚀 Highlights
- Excel-based business rules authoring with instant REST API deployment
- Type-safe validation, version control, and hot reload for zero-downtime updates
- Comprehensive testing, audit trails, and enterprise-grade governance
- Multi-language support: Java, Python, JavaScript, .NET, Go, and more
- Proven performance and scalability for Fortune 500 production workloads
