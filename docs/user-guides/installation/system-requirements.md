## System Requirements

### Operating Systems
- **Windows**: 11+
- **Linux**: Ubuntu 22.4+
- **macOS**: 15+

### Supported Browsers
- **Microsoft Edge**: 131+
- **Firefox**: 128 ESR+
- **Chrome**: 131+

### Databases
- **MySQL**: 8+
- **MariaDB**: 10.5+
- **MS SQL Server**: 2014+
- **Oracle**: 12c+
- **PostgreSQL**: 11.2+

### Other Software
- **Java**: OpenJDK 11 or 21
- **Application Servers**: Apache Tomcat 9, Jetty 10

### Hardware Requirements
- **RAM**: Minimum 4GB (6GB recommended)
- **Processor**: 1GHz+ (32-bit or 64-bit)
- **Disk Space**: 2GB+ for installation and workspace
