Import Test Cases from JSON

Note: Upload a JSON file containing test cases to add them to a test table in your OpenL project. The JSON file should specify the target table name and the test cases with their parameters and expected results.
Example JSON Format: 
{
  "tableName": "YourMethodName",
  "testTableName": "YourMethodNameTest",
  "appendToExisting": false,
  "testCases": [
    {
      "_id_": "TC001",
      "_description_": "Test case description",
      "parameters": {
        "param1": "value1",
        "param2": 123,
        "param3": true
      },
      "_res_": "expectedResult"
    },
    {
      "_id_": "TC002",
      "parameters": {
        "param1": "value2",
        "param2": 456
      },
      "_res_": "anotherExpectedResult"
    }
  ]
}
