1. Tutorial_1.xls : Intro
Tutorial1 - Intro to Decision Tables This is the first tutorial in a series of tutorials designed to familiarize you with OpenL Tablets step-by-step, taking you from simple features and concepts to more complex ones. These tutorials were designed to be somewhat interactive to make you familiar with OpenL Tablets Eclipse IDE and basic editing actions. It is assumed that you are already familiar with Java development under Eclipse. Each Tutorial is located in a separate project named org.openl.tablets.tutorial[N], where N is tutorial number. It is highly recommended that you do these tutorials in the natural order. Each tutorial contains set of working examples, each lesson will end with Run command. Make sure the run poduces the desired results, otherwise you probably missed something. Good luck and have fun
